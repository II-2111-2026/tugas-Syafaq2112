# Laporan Quarto (opsional, ramah dibaca)

Setiap minggu memiliki file:
- `submissions/wXX/laporan.qmd`

Tujuan:
- Menampilkan hasil dari fungsi di `submissions/wXX/answers.py` dalam bentuk laporan yang rapi.
- **Tidak ada dualisme jawaban**: sumber kebenaran tunggal tetap `answers.py`.

Cara render (lokal):
```bash
quarto render submissions/w01/laporan.qmd
```

Catatan:
- Laporan ini bukan penilaian resmi.
- Penilaian resmi tetap dari `pytest` (autograder).
