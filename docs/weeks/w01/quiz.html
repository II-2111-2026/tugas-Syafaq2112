2.3 Task 01 — At least one head

A fair coin is flipped 3 times. Return P(at least one head).

2.4 Task 02 — Combinations

From 10 students, choose 3 (order does not matter). Return the number of committees.

2.5 Task 03 — Conditional probability

In a class of 40 students: - 22 use Python (P) - 18 use R (R) - 10 use both
Return P(Python | R).

2.6 Task 04 — Bayes

Disease prevalence P(D)=0.01, sensitivity P(+|D)=0.95, specificity P(-|not D)=0.90.
Return P(D | +).

2.7 Task 05 — Expected value

X in {0,1,2,3} with probs [0.1, 0.2, 0.3, 0.4]. Return E[X].

2.8 Task 06 — Binomial PMF

X ~ Binomial(n=4, p=0.3). Return [P(X=0),…,P(X=4)].

2.9 Task 07 — Variance of a fair die

X is uniform over {1..6}. Return Var(X).

2.10 Task 08 — Correlation

Cov(X,Y)=2, Var(X)=9, Var(Y)=16. Return Corr(X,Y).

2.11 Task 09 — Normal probability

X ~ Normal(μ=10, σ=2). Return P(8 < X < 12).

2.12 Task 10 — Margin of error

Known σ=4, n=25, 95% z=1.96. Return margin of error.

2.13 Task 11 — Two-sided p-value

Two-sided z-test with z=2.1. Return p-value.

2.14 Task 12 — Monte Carlo estimate

Estimate P(Z > 1.5), Z~N(0,1) using: - n=200000 - seed=123 Return estimated probability.
